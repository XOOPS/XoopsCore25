Aide : Message privé

DESCRIPTION

Le module PM offre la possibilité aux utilisateurs de s'adresser entre eux des messages privés

INSTALLER / DÉSINSTALLER

Aucune mesure spéciale n'est nécessaire, suivez le processus d'installation standard - extrayez le dossier /pm dans le répertoire ../modules. Installez le module via Administration -> Module Système -> Modules.

Un tutoriel détaillé d'installation des modules dans Xoops est disponible à partir de ce lien Manuel d'utilisation de Xoops

MODE D'EMPLOI

Pour configurer ce module, vous devez :

  • Entrer dans votre boîte de réception directement à partir de la page d'accueil, ou en cliquant sur le lien "Aller au module" dans la zone d'administration de PM.
  • Configurer vos préférences pour le module (voir "Préférences") et éventuellement le bloc PM si vous avez l'intention de l'utiliser (voir "Blocs").
  • Vous pouvez supprimer massivement vos messages à l'aide de l'onglet "Purge des messages".
