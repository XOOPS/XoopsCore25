/****** english ******/
To view the blocks in the cpanel:
- Install the plugin smarty function.block.php (http://www.xuups.com/modules/publisher/item.php?itemid=10) in root/class/smarty/xoops_plugins/
- Copy the file xo_block.tpl to "extras" in "xotpl"
- Remove the comments to activate the template xo_block.tpl
- Edit the template xo_block.tpl and put Id number of blocks you wish to display

/****** french ******/
Pour afficher des blocs dans le cpanel :
- installer le plugin smarty function.block.php (http://www.xuups.com/modules/publisher/item.php?itemid=10) dans root/class/smarty/xoops_plugins/
- copier le fichier xo_block.tpl du dossier "extras" dans "xotpl"
- retirer les commentaires pour activer le template xo_block.tpl
- éditer le template xo_block.tpl et saissisez le numéro des blocs que vous désirez afficher
