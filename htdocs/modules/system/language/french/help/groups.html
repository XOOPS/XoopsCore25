Aide : Groupes

Description

XOOPS intègre un système d'enregistrement des membres - vous pouvez éventuellement demander à des personnes de s'inscrire sur votre site afin d'avoir accès à certaines zones ou fonctions. Par exemple, la plupart des sites requièrent que des personnes s'y inscrivent avant d'être autorisées à contribuer ou à poster des messages sur les forums.
Les droits d'accès et d'administration d'un site internet XOOPS sont définis à l'aide d'un système d'autorisations flexible basé sur des « groupes d'utilisateurs ». Les groupes par défaut incluent les utilisateurs anonymes, les membres enregistrés et les webmestres, mais bien sûr vous pouvez définir autant de groupes que nécessaire pour gérer votre site. Chaque groupe peut disposer d'un ensemble unique de droits d'accès régissant :

- Quels blocs peuvent être vus
- Quels modules peuvent être accessibles
- Quels modules peuvent être administrés
- Quels aspects de l'administration du système peuvent être modifiés

Vous pouvez changer le nom de ces groupes par défaut, mais vous ne pouvez pas les supprimer. D'autres groupes d'utilisateurs peuvent également être créés et définis comme indiqué plus tard.

Les groupes personnalisés peuvent également être créés via la page d'administration des groupes.
