Le système de classement (que nous pourrions également nommer : rang, position, ..) permet de donner par exemple une "hiérarchie" ou une "ancienneté" ou une "fonction" à vos utilisateurs, par défaut il est basé dans XOOPS sur le nombre de contribution sur le contenu du site. le site !

Comme indiqué plus haut, le classement par défaut est basé sur le nombre de contributions. Le classement de l'utilisateur sera actualisé automatiquement dès qu'il aura atteint le nombre de contributions requis pour le classement supérieur. Les rangs par défaut de XOOPS sont: "Débutant", "Aspirant", "Régulier", "Semi Pro", "Accro". Il intègre également deux classements spéciaux: "Modérateur" et "Administrateur". Un classement spécial diffère dans le sens où il peut être attribué sans tenir compte des critères requis pour le classement de base (dans la cas de XOOPS - sur les contributions). Une utilisation typique de classement spécial, est de donner par exemple une reconnaissance à des donateurs ou des utilisateurs ayant des functions.

Le lien «modifier» vous permet de modifier le nom d'un classement, de modifier le nombre minimum / maximum de publications nécessaires pour le qualifier, de le désigner comme un classement «spécial» (ou non) et de télécharger une nouvelle image de classement. Vous pouvez supprimer les rangs inutiles (lien 'supprimer').

De même, vous pouvez ajouter un nombre illimité de classement suivant vos besoins. Pour cela, il vous suffira de renseigner les champs du formulaire de création.
