Aide : Publipostage

Description

XOOPS dispose d'un système de publipostage intégré permettant d'envoyer un message aux utilisateurs enregistrés sur le site.
L'envoi s'effectue soit sous la forme d'un message privé (via le système de messagerie intégrée) soit par courrier électronique.
Des filtres sont disponibles afin de n'envoyer un message qu'à un groupe souhaité de membres.

Il existe des balises prédéfinies qui pourront personnaliser le message.
