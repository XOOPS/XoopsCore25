Aide : Bannières

Description

Les bannières sont des publicités affichées dans la section supérieure de la page. Il peut s'agir d'images ou de films flash, par ex. la bannière tournante XOOPS qui fait partie de l'installation par défaut. XOOPS a un système intégré pour gérer les bannières et créer des comptes pour les clients publicitaires, qui est accessible dans le Module Système -> Bannières. Vous pouvez avoir plusieurs bannières sur un site XOOPS - la zone d'affichage pivote à tour de rôles sur chacune des bannières actives chaque fois qu'une nouvelle page est chargée.

Chaque fois qu'une bannière est affichée à un visiteur, cela compte comme une « impression », qui est la monnaie de négociation commune des bannières. Les clients paient généralement un certain nombre d'impressions, c'est-à-dire que leur publicité est affichée un certain nombre de fois.

Remarque : Si vous vendez des bannières en fonction du nombre d'impressions, il est important de procéder de la manière suivante : Sélectionnez Module système -> Préférences -> Paramètres généraux et remplissez le champ « Votre adresse IP » (si vous en avez). Cela empêchera vos visites au site d'être inclus dans le nombre d'impressions. Ne pas le faire pourrait être considéré comme quelque peu malhonnête (à noter, cependant, que de nombreux administrateurs n'ont pas une IP permanente - par exemple ceux qui utilisent des modems d'accès commuté).
