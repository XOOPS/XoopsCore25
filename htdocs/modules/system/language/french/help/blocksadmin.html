Aide : Blocs

Description

Les blocs sont des éléments qui peuvent accueillir toutes sortes de contenu (texte, images, liens...).
Ils peuvent être créés et personnalisés ou bien être issus de modules installés.
Ils peuvent être placés n'importe où dans un thème et être clonés et modifiés.
La page d'administration des blocs est l'endroit où tous les blocs sont configurés et administrés.

Les blocs disposent de plusieurs paramètres que l'on peut configurer aisément.
Les bloc issus de modules peuvent avoir des paramètres supplémentaires.

Chaque module installé enregistre ses blocs dans le système, et le groupe des webmestres peut décider :
- Dans quelle(s) partie(s) de pages du site le bloc apparaîtra (Module(s)),
- Dans quelle zone (Colonne, Haut, Bas, Gauche, Droite),
- Quel(s) groupe(s) sera (seront) autorisé(s) à visualiser le bloc.

Tips

Vous pouvez masquer le titre qui s'affichera lorsque le bloc sera affiché en ajoutant "// " (c'est-à-dire slash-slash-espace) comme trois premiers caractères du titre. De cette façon, vous pouvez joindre de manière transparente plusieurs blocs pour créer la vue souhaitée.

Vous pouvez créer vos propres blocs de contenu à l'aide du bouton "Ajouter un bloc" button. Vous pouvez créer votre propre contenu en HTML, en texte brut avec des codes XOOPS ou même en PHP. Dans vos blocs personnalisés, le code "{X_SITEURL}" sera remplacé par l'URL de votre site avec une barre oblique finale.
