Aide : Emoticônes

Description

Les émoticônes sont de petits symboles graphiques que les gens peuvent utiliser pour s'exprimer. Ils peuvent être insérés n'importe où dans le texte d'un article, un commentaire ou un message à la fois du côté de l'utilisateur et du côté administrateur du système. XOOPS est livré avec une collection de 17 émoticônes intégrés. Certains d'entre eux sont affichés dans les formulaires de soumission dans divers modules (y compris les forums, les nouvelles, les téléchargements, les liens et autres). En cliquant sur un émoticône, vous l'insérerez dans le message. Les smileys peuvent également être insérés en tapant leurs ‘codes’ uniques (raccourcis clavier), par exemple en tapant ':' '-' 'D' dans un message, cela insérera cette émoticône :

La liste complète des émoticônes disponibles sur votre système et leurs codes est accessible via les émoticônes, où vous vous trouvez maintenant. Ici vous pouvez ajouter, modifier et supprimer des émoticônes, et contrôler ceux qui sont affichés sur les formulaires de soumission (seulement neuf des 17 sont affichés par défaut)

Si vous ne voulez pas d'émoticônes sur votre système, vous pouvez soit décocher la case "Afficher dans le formulaire ?" (Dans ce cas, ils n'apparaîtront pas sur les formulaires de soumission, mais seront toujours disponibles via leurs codes), ou utiliser le lien de suppression pour les supprimer entièrement.
