Aide : Maintenance

Description

La maintenance fournit plusieurs fonctions de base pour vous aider à maintenir votre installation XOOPS et la maintenir en bonne santé :

- Suppression du dossier cache
- Vidage de la table de sessions
- Réalisation de la maintenance des tables de la base de données (analyser, vérifier, optimiser et réparer)

Vous pouvez également effectuer une sauvegarde de toute votre base de données, de certaines tables uniquement, ou de celles d'un module particulier.
